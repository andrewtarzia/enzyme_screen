# Setup (Ubuntu 16.04) using conda and pip
- Install Anaconda in standard way
- Packages required outside of what comes with CONDA
    - cirpy:
        - pip install cirpy
    - RDKit:
        - conda install -c conda-forge rdkit
    - pubchempy:
        - pip install pubchempy
    - molvs:
        - pip install molvs
